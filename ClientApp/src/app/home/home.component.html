Marítima Dominicana S.A.S.

Evaluación práctica

I- Realice un programa que efectúe la gestión de artículos en un almacén bajo las siguientes especificaciones:

  • Capturar los datos generales del almacén (descripción, ubicación, capacidad de artículos, etc.).
  • Capturar los datos generales de los artículos (descripción, diferentes precios, etc.).
  • Manejo de inventario (no es necesario que sea complejo).
  • Reporte de movimiento de artículos (entrada/salida).

a) Restricciones y permisos:

  • Puede utilizar cualquier tecnología front-end de su preferencia.
  • No puede utilizar scaffolding.
  • Debe utilizar Asp.Net MVC / Core como tecnología principal.
  • Debe utilizar Entity Framework / Core para el acceso a datos.
  • Debe utilizar MS-SQL como motor de base de datos.

b) Puntos a evaluar:

  • Estructura y manejo de base de datos.
  • Manejo del lenguaje de programación.
  • Separación de conceptos.
  • Buenas prácticas.
  • Buenas validaciones.
  • Interfaz gráfica.

c) Entregables:

  • El proyecto desarrollado.
  • README.md con la descripción general, tecnologías implementadas y consideraciones para iniciar el proyecto.
  • Se debe colocar en un repositorio público de su preferencia (preferiblemente GITHUB), y enviar el link a la dirección del correo que proporciona el examen.
  • Fecha de entrega 1 semanas contando a partir de la fecha de entrega.
